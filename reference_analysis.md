# [프로젝트 제안서] LSTM 수요 예측 및 선형계획법(LP) 기반 재고 관리 최적화 시스템

## 1. 프로젝트 개요
본 프로젝트는 과거 판매 데이터를 바탕으로 미래의 수요를 정교하게 예측하고(LSTM), 이를 기반으로 총 운영 비용을 최소화하는 최적 발주량을 산출(Linear Programming)하는 **'데이터 기반 지능형 의사결정 지원 시스템'** 구축을 목표로 합니다. 학교 수업에서 다룬 경영과학(Management Science) 이론과 AI 기술을 실무 데이터에 접목하는 프로젝트입니다.

## 2. 배경 및 필요성
* **기존 방식의 한계:** 유통 현장에서의 주관적 발주는 재고 과다로 인한 유지비 증가나 품절로 인한 기회 손실을 초래합니다.
* **데이터 기반 의사결정:** 시계열 데이터에 내재된 계절성과 추세를 AI로 학습하여 불확실성을 통제하고, 수리적 최적화를 통해 경영 효율성을 극대화할 필요가 있습니다.
* **학문적 융합:** 딥러닝(LSTM)의 예측력과 경영과학의 최적화 기법을 결합하여 분석의 완결성을 높이고자 합니다.

## 3. 목표
* **고정밀 수요 예측:** Kaggle의 Store Item Demand 데이터를 활용하여 아이템별 미래 수요를 예측하며, 오차율($SMAPE$)을 최소화합니다.
* **비용 최소화 모델 구축:** 재고 유지비, 발주비, 품절 손실비를 변수로 하는 목적함수를 설정하고 선형계획법(LP)을 통해 최적 해를 도출합니다.
* **효과 검증:** 전통적인 재고 관리 방식 대비 AI 기반 시스템 도입 시의 예상 비용 절감 효과를 수치로 증명합니다.

## 4. 주요 기능 및 특징
* **LSTM 기반 시계열 모델링:** 다변량 데이터의 시퀀스 패턴을 학습하여 상점/아이템별 단기 및 중기 수요를 예측합니다.
* **경영과학 기법의 결합:** 예측된 수요값을 상수로 사용하여 실제 비즈니스 제약 조건(창고 용량, 리드 타임 등) 하에서의 최적화 문제를 해결합니다.
* **시나리오 시뮬레이션:** 다양한 비용 구조 시나리오에 따른 결과 비교 분석 기능을 제공합니다.

## 5. 기대 효과
* **운영 효율성 제고:** 최적 발주량 산출을 통해 불필요한 재고 유지 비용과 품절 손실을 동시에 억제할 수 있습니다.
* **의사결정 객관화:** 주관적 판단이 아닌 데이터와 수리 모델에 근거한 객관적인 지표를 제시합니다.
* **실무 역량 강화:** 데이터 분석부터 수리적 최적화까지 이어지는 통합적인 문제 해결 능력을 확보합니다.

# [참고 프로젝트 분석 내용]

### 1. 참고 프로젝트 개요
* **주제:** Kaggle - Store Item Demand Forecasting Challenge
* **목표:** 10개 상점, 50개 품목의 5년 치 일별 판매 데이터를 분석하여 향후 3개월간의 수요를 예측하는 시계열 경진대회입니다.

### 2. 선택 이유
* **데이터의 신뢰성:** 5년간의 방대한 일별 데이터가 축적되어 있어 LSTM 같은 딥러닝 모델이 패턴을 학습하기에 충분한 규모를 가집니다.
* **학습 도메인 연계:** 단순히 예측에서 끝나는 기존 과제를 넘어, 현재 학습 중인 경영과학의 최적화 모델을 덧붙여 실제 비즈니스 문제 해결 프로세스로 확장하기에 최적인 데이터셋입니다.

### 3. 해당 프로젝트 분석 내용
* **데이터 특성:** 주 단위의 반복적인 패턴(요일 효과)과 연 단위의 계절성(Seasonality)이 뚜렷하게 나타나는 특징이 있습니다.
* **평가 척도:** 예측값과 실제값의 차이를 비율로 계산하는 $SMAPE$를 주요 평가지표로 활용합니다.
$$SMAPE = \frac{100\%}{n} \sum_{t=1}^n \frac{|\hat{y}_t - y_t|}{(|\hat{y}_t| + |y_t|)/2}$$
* **핵심 분석 과제:** 품목별, 상점별로 상이한 판매 추이를 모델 하나가 얼마나 일반화하여 포착하느냐가 관건이며, 이를 위해 Lag feature(과거값)와 Rolling window(이동 평균) 등의 피처 엔지니어링이 필수적임을 확인하였습니다.
