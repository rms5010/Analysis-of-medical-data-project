# [프로젝트 제안서] 이커머스 물류 최적화를 위한 수요 예측 및 배송 비용 최소화 시스템

## 1. 프로젝트 개요
본 프로젝트는 브라질의 대형 이커머스 플랫폼인 **Olist의 실제 주문 데이터**를 활용하여, 제품 카테고리 및 지역별 미래 수요를 정교하게 예측하고(LSTM), 이를 기반으로 물류 네트워크의 배송 비용과 재고 유지 비용을 최소화하는 **'지능형 물류 의사결정 지원 시스템'** 구축을 목표로 합니다.

## 2. 배경 및 필요성
* **복잡한 물류 구조의 한계:** 이커머스 환경에서는 제품의 무게, 부피, 고객과의 거리 등 다양한 변수가 물류 비용에 복잡하게 얽혀 있습니다. 단순한 직관에 의존한 재고 배분은 막대한 물류비용과 배송 지연을 초래합니다.
* **데이터 통합 역량의 중요성:** 9개의 관계형 테이블(CSV)로 구성된 실제 비즈니스 데이터를 SQL 없이 오직 Python(Pandas)으로 통합하고 정제하는 과정을 통해 실무 수준의 데이터 엔지니어링 및 분석 능력을 검증하고자 합니다.
* **예측과 최적화의 융합:** AI를 통한 단순 '현상 예측'을 넘어, 경영과학(Management Science)의 최적화 기법을 적용해 실제 비즈니스 가치(Cost Reduction)를 창출하는 과정을 시뮬레이션합니다.

## 3. 목표
* **Python 기반 데이터 통합 파이프라인 구축:** Pandas만을 활용하여 주문(Orders), 아이템(Items), 제품(Products), 지리 정보(Geolocation) 등 다차원 데이터를 병합(Merge)하고, 시계열 분석에 적합한 형태로 전처리합니다.
* **카테고리별 시공간 수요 예측:** LSTM(Long Short-Term Memory) 모델을 사용하여 지역별/카테고리별 판매 추이를 학습하고 미래 수요를 예측하며, 오차율($SMAPE$)을 최소화하는 모델을 설계합니다.
* **물류 네트워크 비용 최적화:** 예측된 수요 데이터를 경영과학의 '수송 문제(Transportation Problem)' 모델에 입력하여, 총 배송 비용 및 재고 유지비를 최소화하는 최적의 재고 거점 및 발주 전략을 도출합니다.

## 4. 주요 기능 및 특징
* **다변량 시계열 피처 엔지니어링:** 제품의 물리적 특성(무게, 부피), 위치 정보(Zip code), 시간적 특성(요일, 계절성)을 결합한 고차원 피처를 생성하여 예측력을 극대화합니다.
* **비대칭 손실 함수(Asymmetric Loss) 고려:** 단순 정확도뿐만 아니라, '품절' 발생 시의 기회 손실 비용과 '재고 과잉' 시의 유지 비용을 차등 설계하여 실제 비즈니스 환경에 최적화된 해를 구합니다.
* **Python 기반 통합 분석 환경:** 데이터 로드부터 모델링, 수리 최적화(PuLP 등 활용)까지 전 과정을 Python 단일 환경에서 구현하여 분석의 일관성과 재현성을 확보합니다.

## 5. 기대 효과
* **운영 비용의 정량적 절감:** 최적화 모델 적용 전후의 비용 시뮬레이션을 통해 구체적인 비용 절감액($\%$)을 산출하여 데이터 기반 의사결정의 효용성을 증명합니다.
* **의사결정 프로세스의 객관화:** 복잡한 물류 변수를 수리 모델로 처리함으로써, 실무자의 주관을 배제하고 데이터에 근거한 객관적인 발주 및 배송 가이드를 제시합니다.
* **실무형 포트폴리오 확보:** 원천 데이터 가공부터 최적화 알고리즘 적용까지 이어지는 엔드투엔드(End-to-End) 프로젝트 수행 역량을 입증합니다.

# [참고 프로젝트 분석 내용]

### 1. 참고 프로젝트 개요
* **주제:** Brazilian E-Commerce Public Dataset by Olist
* **목표:** 브라질 이커머스 플랫폼 Olist의 10만 건 이상의 실제 주문 데이터를 분석하여 구매 패턴, 배송 성능 등을 파악하는 비즈니스 데이터셋입니다.
* **참고 링크:** [Kaggle - Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

### 2. 선택 이유
* **현장감 있는 데이터 구조:** 9개의 CSV 파일이 관계형으로 연결되어 있어, 이를 Python으로 직접 병합하고 정제하며 실무적인 데이터 핸들링 능력을 키우기에 최적입니다.
* **경영과학적 활용 가치:** 지리 정보와 물류비(Freight Value) 정보가 포함되어 있어, 단순 예측을 넘어 경영과학의 최적화 모델(비용 최소화, 수송 최적화)을 적용할 변수가 매우 풍부합니다.

### 3. 해당 프로젝트 분석 내용
* **데이터 특성:** 브라질 전역의 광범위한 배송 지역과 70개 이상의 카테고리를 포함하고 있어 지역별 수요 편차가 크며, 이를 포착하는 것이 분석의 핵심입니다.
* **평가 척도:** 수요 예측에는 비율 기반 오차 지표인 $SMAPE$를 사용하며, 최종 의사결정 단계에서는 '총 물류 운영 비용($Total\,Logistics\,Cost$)'을 핵심 평가지표로 설정합니다.
$$SMAPE = \frac{100\%}{n} \sum_{t=1}^n \frac{|\hat{y}_t - y_t|}{(|\hat{y}_t| + |y_t|)/2}$$
* **핵심 분석 과제:** 제품 무게/부피에 따른 배송비 가중치를 반영하고, 특정 카테고리의 수요 변동에 대응하는 탄력적 재고 관리 모델을 설계하는 것이 본 프로젝트의 차별점입니다.
